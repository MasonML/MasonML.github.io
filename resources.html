Resources

Blogs
An overview of gradient descent optimization algorithms
Sebastian Ruder

Good explanations of optimization algorithms that expand upon the traditional gradient descent method. Covers Momentum, Adagrad, Adadelta, and various other methods.

Numerical Optimization: Understanding L-BFGS
aria42

An overview of how the L-BFGS works for unconstrained function minimization.

Books
Neural Networks and Deep Learning
Michael Nielsen

A hands-on, principle-oriented approach to learning core concepts of neural networks and modern techniques for deep learning.

An Introduction to Statistical Learning
Daniela Witten, Gareth James, Robert Tibshirani, Trevor Hastie

A simpler more application focused version of "The Elements of Statistical Learning"

Deep Learning
Ian Goodfellow, Yoshua Bengio, and Aaron Courville

Introduces basic mathematical concepts such as linear algebra, probability and information theory, numberical computation. Describes the fundamentals of machine learning, the state of deep learning, and research in machine learning.

Papers
An introduction to Toplogical Data Analysis: fundamental and practical aspects for data scientists
Frederic Chazal and Bertand Michel

Brief introduction to basic fundamental and practical aspects of Topological Data Analysis (TDA) for non experts.

Why Momentum Really Works
Gabriel Goh

An interactive research paper that explains the Momentum optimization technique in detail with visualizations

Efficient Reinforcement Learning through Evolving Neural Network Topologies
Kenneth Stanley, Risto Mikkulainen

The research paper that introduced the NEAT (Neuroevolution of Augmenting Topologies) genetic algorithm for evolving artificial neural networks.

Efficient Reinforcement Learning through Evolving Neural Network Topologies (Extended)
Kenneth Stanley, Risto Mikkulainen

A longer version of the original paper.

ImageNet Classification with Deep Convolutional Neural Networks
Alex Krizhevsky, Geoggrey Hinton, Ilya Sutskever

A neural network project that classified 1.2 million images in the ImageNet LSVRC-2010 contest into 1000 different classes.

An Introduction to the Conjugate Gradient Method Without the Agonizing Pain
Jonathan Richard Shewchuk

Explains the Conjugate Gradient Method with sixty-six illustrations.

A Perfect Example for The BFGS Method
Yu-Hong Dai

Applies the BFGS quasi-Newton method to a general non-convex function that has continuous second derivatives.

Generative Adversarial Nets
Ian J. Goodfellow, Jean Pouget-Abadie, Mehdi Mirza, Bing Xu, David Warde-Farley, Sherjil Ozair, Aaron Courville, Yoshua Bengio

Applying a framework which uses an adversarial process to estimate generative models.
